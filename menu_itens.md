# Tipos de Ataque / Fases Ofensivas

| Fase                     | Descrição                                                               |
|--------------------------|-------------------------------------------------------------------------|
| Reconhecimento           | Coleta passiva de informações sem interação direta com o alvo.          |
| Varredura (Scan)         | Descoberta de portas, hosts ativos e serviços expostos.                 |
| Enumeração               | Coleta ativa de dados dos serviços descobertos (versão, banners, etc).  |
| Exploração               | Execução de exploits para explorar vulnerabilidades conhecidas.         |
| Força Bruta              | Tentativas automatizadas de autenticação (brute-force/dicionário).      |
| Pós-Exploração (Opcional)| Coleta de informações após acesso obtido (privesc, pivot, etc).         |
| Automatização Completa   | Executa múltiplas etapas encadeadas automaticamente.                     |
| Geração de Relatório     | Resumo e evidência das etapas executadas.                               |
